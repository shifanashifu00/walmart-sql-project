# Walmart SQL Project
## Project Overview

This project analyzes Walmart sales data using MySQL Workbench. The objective is to answer business-related questions using SQL queries and gain insights from sales transactions.

## Tools Used

* MySQL Workbench
* SQL

## Dataset Features

* Invoice ID
* Branch
* City
* Category
* Unit Price
* Quantity
* Date
* Time
* Payment Method
* Rating
* Profit Margin

## Project Objectives

* Analyze sales performance
* Identify branch-wise transactions
* Calculate revenue and profit metrics
* Perform business-oriented SQL analysis

## Skills Demonstrated

* SQL Queries
* Aggregate Functions
* GROUP BY
* ORDER BY
* Filtering Data
* Business Data Analysis
